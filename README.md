# Lagersystem frontend

This project is built using Vue.

### Get started
Install [Node 20 LTS](https://nodejs.org/en/download).

```sh
# you need to have pnpm installed
npm install -g pnpm

# install dependencies
pnpm i

# run dev server
pnpm dev
```
